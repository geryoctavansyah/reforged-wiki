# PATCH RELEASE-1.2-21 NOTES

_Rule Your Own Town and Nation_

## MID-PATCH UPDATES

### Penambahan

- Penambahan NPC Traveller di Dwipantara yang berfungsi untuk memberikan menu untuk pilihan dunia.
- Penambahan Vote Party Crate dan Premium I Crate Box.
- Initial build sistem GPS, yang bisa diakses dengan command /navigator di world Dwipantara.
- Penambahan command /vote untuk melihat link vote.
- Initial build Quest Tutorial untuk pengguna baru.

### Perbaikan

- Perbaikan default channel dari Nation menjadi Local.
- Perbaikan Armor Mage di Blacksmith.
- Perbaikan `SKILL_CRITIAL_POWER` dan `SKILL_CRITICAL` yang sebelumnya tidak akfif.
- Perbaikan chunk error pada saat unclaim chunk.

### Perubahan

- Perubahan menu worlds sekarang sudah bisa langsung akses menuju world nether dan world the end.

- Perubahan mana regen untuk class Assasins dari `1.0` menjadi `0.08`.
- Perubahan mana regen untuk class Archer dari `1.0` menjadi `0.1`.
- Perubahan mana regen untuk class Mage dari `1.0` menjadi `0.3`.
- Perubahan mana regen untuk class Shaman dari `1.0` menjadi `0.25`.
- Perubahan chat, tab, dan tag format dengan adanya data town and nation.
- Perubahan Blacksmith sekarang memerlukan beberapa items vanilla untuk crafting.
- Perubahan peningkatan curva exp dimulai dari level 40.
- Perubahan semua tools vanilla menjadi tools rpg.
- Perubahan semua mobs vanilla menjadi mobs rpg.
- Perubahan coin yang didapet dari setiap kill mobs.
- Perubahan buff harga hasil farming dan hasil mining.

- Penghapusan command /worlds dan perlu menggunakan NPC Traveller yang ada di spawn point Dwipantara untuk mengakses menu worlds.
