# Classes

## Items Restrictions

By default all classes are allowed to use items from `Categories` below. Go to [categories](/items/categories) to learn more.

| Classes  | Categories        |
| -------- | ----------------- |
| Archer   | `Medium`, `Light` |
| Assassin | `Medium`, `Light` |
| Mage     | `Light`           |
| Shaman   | `Heavy`, `Medium` |
| Warrior  | `Heavy`           |
