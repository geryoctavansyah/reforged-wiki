# Commands

## General Commands

| Command                | Description                                           | Options                                   |
| ---------------------- | ----------------------------------------------------- | ----------------------------------------- |
| /channel `<channel>`   | Choose a channel chat                                 | `local`, `town`, `nation`, `mod`, `admin` |
| /message `<username>`  | Send message to specify player                        |                                           |
| /reply `[username]`    | Reply message to specify player                       |                                           |
| /discord               | Send discord invite link                              |                                           |
| /discord link          | Link your discord to minecraft account                |                                           |
| /discord linked        | Check is your minecraft account linked to any discord |                                           |
| /discord unlink        | Unlink your discord to minecraft account              |                                           |
| /spawn                 | Teleport to spawn point                               |                                           |
| /back                  | Back to your latest teleported                        |                                           |
| /menu                  | Open menu main                                        |                                           |
| /navigator             | Open menu navigator                                   |                                           |
| /store                 | Open menu store                                       |                                           |
| /vote                  | Send vote link                                        |                                           |
| /skin                  | Send message for skin help                            |                                           |
| /skin set `<username>` | Change skin using username                            |                                           |
| /skin url `<url>`      | Change skin using url                                 |                                           |
| /skin update           | Update skin to latest version                         |                                           |
| /skin clear            | Clear skin                                            |                                           |
| /ignore                | Ignore player                                         |                                           |
